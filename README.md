A prototype robot capable of detecting and following a human using sensor-based navigation and embedded control systems.

## Overview

The Human Following Robot is an autonomous robotic system designed to track and follow a person while maintaining a safe distance. The robot processes sensor inputs and controls motor movement in real time.

## Features

- Human tracking
- Autonomous movement
- Obstacle detection
- Real-time navigation
- Embedded control system

## Hardware Components

- Arduino Nano
- Ultrasonic Sensors
- Motor Driver
- DC Motors
- Robot Chassis
- Battery Pack

## Working

The robot continuously detects a person's position using sensors and adjusts its movement accordingly. The controller processes sensor data and drives the motors to maintain a safe following distance.

## Applications

- Smart Assistance Systems
- Educational Robotics
- Autonomous Navigation
- Service Robots
